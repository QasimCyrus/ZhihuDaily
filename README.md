## 已开发功能：
1. 根据日期查看新闻，或者根据分类专栏来查看新闻，新闻的查看无需跳转到浏览器；
2. 已查看新闻具有自动缓存功能，支持离线查看，同时还可以对新闻进行收藏；
3. 夜间模式的切换，由于使用WebView加载新闻内容，所以前端css方面并不能很完善。
